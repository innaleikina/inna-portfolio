Hangman

×

This is a Hangman game based on cult classic movie titles. The game utilized HTML, CSS and JavaScript. The game picks a random title from an array, after the first wrong guess the user gets a hint containing a quote from the movie. If the movie is guessed correctly an image from the movie comes up. If the movie title is guessed incorectly the answer is given after all 8 tries to guess have been exhausted.
