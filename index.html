APPS

Check out my responsive apps by clicking on a project

The weather app uses the W3C Geolocation API to find the user's location and Free Code Camp API to display accurate temperature and weather conditions. The app also has a button to convert Celsius to Fahrenheit.

The Wikipedia app uses the Wikipedia API to pull up results based on the user's search. The random article button takes the user to a random Wikipedia article.

The Random Quote Generator uses the Quotes On Design API, to generate a random quote each time the user presses a button. I also used the Twitter API to make each quote tweetable.
